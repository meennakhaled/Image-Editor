# Image Editor (C++)

## Overview

Image Editor is a C++ console application.
The program allows users to load **BMP images**, apply multiple filters and transformations, then save the edited result.

Two versions are included:

* **Grayscale Version** → Works on black & white images.
* **RGB Bonus Version** → Works on colored images using Red, Green, Blue channels.

---

## Features

The program supports the following image operations:

1. Black & White Filter
2. Invert Filter
3. Merge Two Images
4. Flip Image (Horizontal / Vertical)
5. Darken / Lighten Image
6. Rotate Image (90 / 180 / 270 / 360)
7. Detect Edges
8. Enlarge Quarter of Image
9. Shrink Image
10. Mirror Half Image
11. Shuffle Image Quarters
12. Blur Image
13. Crop Image
14. Skew Right
15. Skew Up
16. Save Edited Image
